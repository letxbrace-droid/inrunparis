Installer le site client
Seul le Site de réservation dispose d'une PWA installable sur l'écran d'accueil.
  1. 1
    Ouvrir https://letxbrace-droid.github.io/inrunparis/ dans Safari (iOS) ou Chrome (Android).
  2. 2
    iOS : Appuyer sur le bouton Partager puis « Sur l'écran d'accueil ».
  3. 3
    Android : Menu Chrome (trois points) puis « Ajouter à l'écran d'accueil ».
  4. 4
    Confirmer l'installation — l'icône I&N RUN apparaît sur l'écran d'accueil.

# Ad Platform Specs & Creative Guidelines

Quick-reference for platform-specific specs, safe zones, and creative best practices. Use during Phase 3 analysis to evaluate platform compliance.

---

## Meta (Facebook + Instagram)

### Specs by Placement

| Placement | Aspect Ratio | Resolution | Video Length | File Size |
|-----------|-------------|-----------|-------------|-----------|
| Feed (FB + IG) | 1:1 or 4:5 | 1080×1080 min | 1s–60s optimal (15-30s best) | Image 30MB / Video 4GB |
| Stories / Reels | 9:16 | 1080×1920 | Up to 60s (15s optimal) | Image 30MB / Video 4GB |
| Carousel | 1:1 | 1080×1080 | N/A (per card) | Image 30MB per card |
| Right Column (FB) | 1.91:1 | 1200×628 | N/A | Image 30MB |
| Marketplace (FB) | 1:1 | 1080×1080 | N/A | Image 30MB |
| In-Stream Video | 16:9 or 1:1 | 1080p | 5-15s (non-skippable) | Video 4GB |

### Safe Zones — Stories / Reels
```
┌─────────────────────┐
│  TOP 14% (250px)    │ ← Profile icon, name, "Sponsored" label
│  DO NOT PLACE TEXT  │
│─────────────────────│
│                     │
│   SAFE ZONE         │
│   1080 × 1420       │
│   (center area)     │
│                     │
│─────────────────────│
│  BOTTOM 20% (340px) │ ← CTA button, description, swipe-up
│  DO NOT PLACE TEXT  │
└─────────────────────┘
```

### Key Guidelines
- **Text overlay:** No longer rejected for >20% text, but delivery is still penalized. Minimize text on images.
- **Primary text:** 125 characters visible before "See more" truncation. Headline: 27 characters on mobile.
- **Sound:** 85% of feed video watched on mute. Captions are essential.
- **First frame:** Must not be blank or black. The thumbnail determines if the video plays.
- **Advantage+ Creative:** Meta's system automatically tests asset combinations. Design individual components (headline, image, description, CTA) to work in any combination.

### Platform Creative Best Practices
- Lead with the strongest visual. Feed moves fast — 1.7 seconds average attention per post.
- Human faces with eye contact outperform product-only imagery.
- Vertical content (4:5 or 9:16) takes up more feed real estate = more attention.
- Native/authentic content outperforms polished studio content for most objectives.
- Looping short videos (<15s) are highly effective for engagement.

## Google Ads

### Display Network Banner Sizes

| Size | Name | Usage |
|------|------|-------|
| 300×250 | Medium Rectangle | Most widely available. Use as default. |
| 336×280 | Large Rectangle | High-performing desktop format. |
| 728×90 | Leaderboard | Desktop header placement. |
| 300×600 | Half Page | High-impact desktop. |
| 320×50 | Mobile Leaderboard | Standard mobile banner. |
| 320×100 | Large Mobile Banner | Better mobile real estate. |
| 160×600 | Wide Skyscraper | Sidebar placement. |

### Responsive Display Ad Assets

| Asset | Spec | Quantity |
|-------|------|----------|
| Landscape image | 1200×628 (1.91:1) | 1 required, up to 15 |
| Square image | 1200×1200 (1:1) | 1 required, up to 15 |
| Logo (square) | 1200×1200 | 1 required |
| Logo (landscape) | 1200×300 | 1 recommended |
| Headlines | Max 30 characters | Up to 5 |
| Long headline | Max 90 characters | 1 |
| Descriptions | Max 90 characters | Up to 5 |

**Ad Strength grades:** Poor / Average / Good / Excellent. Based on asset diversity, relevance, quantity. To reach "Excellent": provide 5+ headlines, 5+ descriptions, 3+ images, ensure text diversity.

### YouTube Video Ads

| Format | Length | Skip | Aspect Ratio |
|--------|--------|------|-------------|
| Skippable in-stream | 12s–3min recommended | At 5s | 16:9 (standard), 9:16/1:1 for Shorts |
| Non-skippable | 15s or 20s | No | 16:9 |
| Bumper | Max 6s | No | 16:9 |
| Shorts | Up to 60s | At 5s | 9:16 |

**Resolution:** 1920×1080 recommended. **File size:** Max 256GB.

### Google ABCD Framework
- **Attention:** First 5 seconds must hook. Use bold visuals, surprise, direct address, or action.
- **Branding:** Introduce brand early (within 5s) and often. Audio mention + visual logo.
- **Connection:** Build emotional resonance. Human faces, storytelling, humor.
- **Direction:** Clear CTA with urgency. Tell the viewer exactly what to do.

## TikTok

### Specs

| Spec | Value |
|------|-------|
| Aspect ratio | 9:16 (strongly preferred), 1:1 acceptable |
| Resolution | 1080×1920 recommended |
| Video length | 5–60s. **21–34 seconds is the conversion sweet spot.** |
| File size | Max 500MB |
| Image ads | 1200×628 (1.91:1) for lead gen |

### Safe Zones
```
┌─────────────────────┐
│  TOP 130px          │ ← Status bar, navigation
│─────────────────────│
│                     │
│   SAFE ZONE         │
│   ~720 × 1060      │  ← RIGHT 100px reserved for
│   (center area)     │    interaction buttons (like,
│                     │    comment, share, follow)
│─────────────────────│
│  BOTTOM 440px       │ ← Caption text, CTA button,
│                     │    music ticker
└─────────────────────┘
```

### Key Guidelines
- **"Don't make ads. Make TikToks."** Native-feeling content outperforms polished ads 2:1.
- Hook must land in **first 2 seconds**. Text hooks on screen immediately.
- Trending sounds/music increase distribution.
- Feature real people, not animations or stock.
- Fast-paced editing (scene changes every 2-3 seconds).
- **Spark Ads** (boosted organic posts) perform 30-50% better on engagement than traditional in-feed.
- Vertical video only. Landscape content is heavily penalized by the algorithm.

## X (Twitter)

### Specs

| Format | Spec |
|--------|------|
| Single Image | 1200×675 (1.91:1) or 1200×1200 (1:1). Max 5MB. |
| Video | 1280×720 min. 16:9, 1:1, or 9:16. Max 2m15s (15s recommended). Max 1GB. |
| Carousel | 2-6 cards. 800×800 (1:1) or 800×418 (1.91:1). |
| Website Card | Image 800×418 min. Title max 70 characters. |

### Key Guidelines
- First 3 seconds critical for video.
- Persistent logo/brand mark recommended throughout.
- Captions/text overlays boost video completion rate by 28% (X internal data).
- News-hook and trending-topic creative outperforms evergreen content on X.
- Keep copy concise — X's audience expects brevity.

## LinkedIn

### Specs

| Format | Spec |
|--------|------|
| Single Image | 1200×628 (1.91:1) or 1200×1200 (1:1). Max 5MB. |
| Video | 360p–1080p. 16:9, 1:1, 9:16. 3s–30min (30-90s awareness, 15-30s consideration). Max 200MB. |
| Carousel | 2-10 cards. 1080×1080 (1:1). Max 10MB per card. |
| Document Ads | PDF/DOC/PPT. Max 100MB, 10 pages recommended. |

### Text Limits
- Introductory text: 150 characters before truncation (600 max)
- Headline: 70 characters recommended

### Key Guidelines
- Professional tone expected but humanized content outperforms corporate polish.
- Statistics and data points in ad copy drive higher CTR on LinkedIn specifically.
- Employee stories and behind-the-scenes content builds trust.
- Document ads (PDF carousels) are highly effective for B2B lead gen.
- Thought leadership content outperforms direct-response messaging for awareness.

## Pinterest

### Specs

| Format | Spec |
|--------|------|
| Standard Pin | 1000×1500 (2:3) recommended. Max 20MB. |
| Video Pin | 1000×1500 (2:3), 1:1, or 16:9. 4s–15min (6-15s optimal). Max 2GB. |
| Idea Pins | Up to 20 pages. 1080×1920 (9:16). Video clips 1-60s each. |
| Carousel | 2-5 cards. 1:1 or 2:3. 1000×1500 recommended. |

### Key Guidelines
- Text overlays work well on Pinterest (unlike Meta). Pins with text overlay have **23% higher CTR**.
- Vertical format strongly preferred — 2:3 gets the most feed real estate.
- Lifestyle context (product in use) outperforms product-on-white by 30%.
- Clean, aspirational imagery aligns with Pinterest user intent (planning, discovering).
- Pinterest is a search+discovery platform — keyword-rich descriptions matter.

## Text Readability on Mobile

Minimum readable text sizes at 1080px width (approximate — test on actual devices):

| Element | Minimum Size | Recommended |
|---------|-------------|-------------|
| Headline | 40px | 56-72px |
| Subheadline | 28px | 36-48px |
| Body text | 24px | 28-36px |
| Fine print / legal | 18px | 22-24px |
| CTA button text | 28px | 36-48px |

**Rule of thumb:** If you can't read it when the image is displayed at 375px wide (iPhone screen), it's too small. Most ad previews at feed size are 375-428px wide.

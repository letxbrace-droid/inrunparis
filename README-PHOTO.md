# I&N RUN — Intégration de la photo

## Structure du dépôt attendue

```
inrunparis/
├── index.html       ← ce fichier
└── img/
    └── voiture.jpg  ← TA photo ici
```

## Comment ajouter la photo en 3 clics

1. Sur GitHub, ouvre ton dépôt letxbrace-droid/inrunparis
2. Crée le dossier img/ → bouton "Add file" → "Create new file"
   Tape :  img/voiture.jpg  dans le nom (ça crée le dossier automatiquement)
   OU glisse-dépose directement ton fichier dans le dossier img/
3. Valide avec "Commit changes"

## La photo s'affichera automatiquement !
- Si img/voiture.jpg existe  → photo plein écran avec fondu doré
- Si le fichier est absent   → placeholder élégant avec icône 🚗

## Renommer autrement ?
Change juste cette ligne dans index.html :
   src="img/voiture.jpg"
par ex: src="img/toyota.jpg"

## Conseils photo pour rendu premium
- Format paysage 16:9 ou 3:2
- Minimum 1200px de large
- Prise de vue 3/4 avant ou latérale
- Lumière naturelle ou coucher de soleil
- Ton sombre = parfait pour le design dark

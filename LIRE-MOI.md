# Comment déployer

1. Remplacer index.html dans GitHub par celui de ce ZIP
2. Remplacer mentions-legales.html (ou créer si absent)
3. Uploader votre photo de véhicule à la RACINE, nommée: voiture.jpg
4. Compléter votre SIRET dans les 2 fichiers HTML
5. Commit -> site en ligne en 2 minutes !
